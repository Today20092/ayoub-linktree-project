# Redesign

## Preserve

Audit before changing: brand tokens, navigation and information architecture, copy voice, top pages, conversion paths, analytics, search metadata, structured data, and existing accessible behavior. State what remains stable and why. Change the minimum needed to improve the stated problem.

## Overhaul

Retain content and routes unless the brief authorizes broader changes. Map old content to the new hierarchy before deleting sections. Preserve semantic headings, metadata, canonical URLs, performance budgets, and essential interactions. Make irreversible content or route changes explicit before executing them.

## Both modes

Compare the before and after at the key viewport widths. Confirm that visual changes have not reduced findability, contrast, page speed, or the clarity of the primary action.

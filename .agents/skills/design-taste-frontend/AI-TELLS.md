# Avoid Templated Signals

| Instead of | Do this |
| --- | --- |
| A gradient as the only brand expression | Build a palette, type, image, and composition decision that fit the subject |
| Three equal feature cards | Use the hierarchy the content earns: an editorial sequence, a proof strip, a comparison, or a focused feature |
| A tiny uppercase label above every heading | Reserve eyebrows for true navigation or context |
| Generic aspirational copy | Name the subject, audience, outcome, place, craft, or proof |
| Floating glass panels without product purpose | Use surfaces to group controls, evidence, or information |
| A hero made of decorative UI fragments | Use real work, photography, illustration, or a meaningful artifact |
| Uniform rounded corners everywhere | Apply the chosen shape lock selectively and consistently |
| Motion on every scroll entrance | Animate only hierarchy, continuity, feedback, or a deliberate moment |

Audit the page at thumbnail size and at full size. If its structure could belong to any company, strengthen the Design Read decisions rather than adding ornament.
